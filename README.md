本库是尝试复现 lodash 库的实现。

[文档（中文）](https://www.lodashjs.com/docs/latest)
[文档](https://lodash.com/)
[github](https://github.com/lodash/lodash)

基于 ES5 和 ES6 构建，测试暂时用的是 node 的 assert，当然测试环境就是 node，且不包含浏览器环境，之后可能会更换测试框架吧！！！

```shell
# lodash

```
